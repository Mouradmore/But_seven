# Dev Community Platform 🚀

A comprehensive full-stack web application designed to empower developers to write, run, and share their web projects (HTML, CSS, JavaScript) directly from the browser. 

## ✨ Key Features
* **Browser-Based Code Editor:** A fully integrated editing environment utilizing CodeMirror, featuring syntax highlighting, dark mode support, and automatic bracket closure for a seamless coding experience.
* **Community Interactions:** Users can discover the most viewed projects, drop comments, and like their favorite creations.
* **User Authentication:** Secure login and registration system using JWT (JSON Web Tokens) and password encryption.
* **Project Management:** Developers can easily save, edit, preview, and download their custom projects.
* **Full-Stack Architecture:** Built with a Node.js and Express backend, connected to a MongoDB database, and paired with a responsive custom frontend.

## 🛠️ Tech Stack
* **Frontend:** HTML, CSS, JavaScript, CodeMirror
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Authentication:** JWT & bcryptjs
